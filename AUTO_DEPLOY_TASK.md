# Настройка автоматического деплоя

Из-за ограничений PowerShell с кириллицей, я не могу автоматически запускать bat-файлы.

## Решение: Планировщик задач Windows

Можно настроить автоматический запуск `git-push-now.bat` через планировщик задач.

## Ручной запуск (рекомендуется)

Просто двойной клик по `git-push-now.bat` когда нужно отправить изменения.

## На будущее

Я буду всегда создавать bat-файлы для деплоя после изменений, и вы сможете запускать их двойным кликом.
