# Папка assets/ — CTA-элементы

Поместите сюда файлы анимированных элементов для наложения на видео.

## Поддерживаемые форматы
- GIF (анимированные) — рекомендуется
- PNG (статичные, с прозрачностью)
- APNG (анимированные PNG)
- JPG, BMP, WebP

## Рекомендуемый набор элементов

### Руки и указатели
- `hand_point_right.gif` — анимированная рука, указывающая вправо
- `hand_point_down.gif` — анимированная рука, указывающая вниз
- `tap_finger.gif` — анимация нажатия пальцем

### Стрелки
- `arrow_right.gif` — анимированная стрелка вправо
- `arrow_down.gif` — анимированная стрелка вниз
- `swipe_up.gif` — анимация свайпа вверх

### Кнопки и надписи
- `subscribe_button.png` — кнопка «Подписаться» (красная, YouTube-стиль, ~300×80 px, PNG с прозрачным фоном)
- `click_here.png` — надпись «Нажми сюда» (PNG с прозрачным фоном)

### Реакции
- `like_heart.gif` — анимированное сердце
- `thumbs_up.gif` — палец вверх
- `bell_notification.gif` — колокольчик уведомлений

### Эффекты
- `circle_highlight.gif` — анимированный круг-подсветка

## Где скачать
1. **Giphy** — giphy.com (ищите «pointing hand transparent», «subscribe button gif»)
2. **LottieFiles** — lottiefiles.com (конвертируйте Lottie → GIF)
3. **Flaticon / Freepik** — flaticon.com (PNG-иконки)
4. **Canva** — canva.com (создайте свои элементы, экспорт в GIF/PNG)
5. **Tenor** — tenor.com (GIF-анимации)

## Требования к файлам
- Прозрачный фон (для GIF и PNG)
- Разрешение: 200–600 px по большей стороне
- Размер файла: до 5 МБ (для производительности превью)
- GIF: 10–30 кадров, зацикленные

## Добавление через приложение
Вы также можете загрузить файлы через кнопку «Загрузить свой файл» в библиотеке элементов — файл будет автоматически скопирован в эту папку.
